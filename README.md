# Codacy Styleguide

## Use

### Install Dependencies

```sh
yarn
yarn bolt:install
```

### Develop

#### React Components

```sh
yarn build:components
```

#### Stylesheets

```sh
yarn build:stylesheets
```

#### React Website

```sh
yarn watch:website
```

#### Legacy Website

```sh
yarn watch:website-legacy
```

#### Others

Check section `scripts` in [package.json](./package.json)
